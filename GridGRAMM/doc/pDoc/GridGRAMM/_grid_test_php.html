test Grid::initialize()

Grid::initialize() allows us to identify ourselves to the Grid. Identification must be made from a UI node, hence we need to connect first to a UI node.

In later versions, initialize() detects if we are already connected and if not, opens the connection implicitly for us. Both scenarios should be checked.
